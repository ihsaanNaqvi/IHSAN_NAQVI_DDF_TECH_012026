## 1. Objective

This project is a **Proof of Concept (PoC)** demonstrating how the Dadosfera platform can be used to build an end-to-end **Data Platform for an e-commerce company**, delivering descriptive and prescriptive analytics with lower cost and faster time-to-value.

The solution follows the **Dadosfera Data Lifecycle**, from **Data Integration** to **GenAI-powered analytics and Data Apps**.

---

## 2. Business Context

The client is a large e-commerce company aiming to:
- Centralize multiple data sources
- Improve decision-making across business areas
- Extract value from unstructured product data
- Enable scalable analytics and AI use cases

---

## 3. Dataset Overview

A **synthetic e-commerce dataset** was generated to represent:
- Product catalog (structured + unstructured text)
- Sales transactions
- Temporal and categorical dimensions

**Total records:** 150,000+  
**Domain:** Products, Sales, Categories, Regions

## 4. Project Structure

Each folder corresponds to a stage of the Dadosfera Data Lifecycle:

| Folder | Description |
|------|------------|
| 00_project_planning | Project planning, risks, and scope |

| 07_analysis_dashboards | SQL analysis and dashboards |
